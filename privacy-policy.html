Privacy Policy

This Privacy Policy describes how GSC URL Insights (“the extension”) handles user information.

Information Collection

GSC URL Insights does not collect, store, or transmit any personally identifiable information to external servers.

The extension accesses Google Search Console data only after the user explicitly signs in with their Google account and grants permission.

Use of Google APIs

GSC URL Insights uses the Google Search Console API to retrieve search performance metrics (such as clicks and impressions) for the currently opened webpage.

Access to Google data is read-only and strictly limited to what is necessary for displaying page-level insights.

Data Usage

  • All data is processed locally within the user’s browser.
  • No Google Search Console data is stored permanently.
  • No data is shared with third parties.

Local Storage

The extension may use Chrome’s local storage to temporarily store authentication status in order to improve user experience and avoid repeated sign-in prompts.

Third-Party Services

The extension does not include analytics, advertising, or tracking services.

User Control

Users can revoke the extension’s access to their Google account at any time through their Google Account permissions.

Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected in the latest version of the extension.

Contact

If you have questions or concerns about this Privacy Policy, please contact the extension developer through the Chrome Web Store listing.
