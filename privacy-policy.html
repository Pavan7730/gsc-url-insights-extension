Privacy Policy

Last updated: 27 January 2026

GSC URL Insights (“the Extension”) is a Chrome browser extension that allows users to view Google Search Console performance data (such as clicks, impressions, and search queries) for the currently opened webpage.

Your privacy is important to us. This Privacy Policy explains what data is accessed, how it is used, and how it is protected.

1. Data We Access

With your explicit permission, the Extension accesses the following data from your Google Search Console account:

  • Page-level performance metrics (clicks, impressions, CTR, average position)
  • Search queries (keywords) associated with a page
  • Date-range based performance data

The Extension uses the Google Search Console API with read-only access.

No data is modified, deleted, or submitted back to Google Search Console.

2. How Data Is Used

All accessed data is used solely to display insights inside the Chrome extension interface.

The Extension does not use data for advertising, tracking, profiling, or analytics purposes.

3. Data Storage

  • OAuth access tokens are stored locally in the user’s browser using Chrome secure storage.
  • No user data is transmitted to external servers.
  • The Extension does not operate any backend server.

4. Data Sharing

The Extension does not sell, rent, or share user data with third parties.

All communication occurs directly between:

  • The user’s browser
  • Google’s official Search Console API

5. Permissions Explanation

The Extension requests the following permissions:

  • Google Account access (OAuth): Used to authenticate users securely.
  • Search Console Read-only Scope: Allows viewing Search Console performance data only.
  • Tabs permission: Used to detect the currently opened webpage URL.
  • Storage permission: Used to store OAuth tokens locally in the browser.

6. User Control

Users may:

  • Revoke access at any time via their Google Account permissions page
  • Remove the extension to delete all locally stored data
  • Choose which Google account is used for authentication

7. Children’s Information

This Extension is not intended for use by children under the age of 13 and does not knowingly collect personal information from children.
