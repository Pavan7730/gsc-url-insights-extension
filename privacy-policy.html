Privacy Policy for GSC URL Insights

Last updated: February 2026

GSC URL Insights is a Chrome extension designed to display basic information about the currently opened webpage and, in future updates, Google Search Console insights related to that page.

Information Collection and Use

This extension does not collect, store, transmit, or sell any personally identifiable information.

The extension may access the URL of the currently active tab solely to display page-specific information inside the extension popup. This data is processed locally in the user's browser and is not sent to any external servers controlled by the developer.

Google APIs

In future updates, the extension may allow users to authenticate with their own Google account to access Google Search Console data via official Google APIs.

Any authentication and data access will be handled securely using Google’s OAuth system. The developer does not store or have access to users’ Google account credentials or Search Console data.

Data Sharing

The extension does not share any user data with third parties.

Data Security

All processing occurs locally within the browser. No user data is stored or transmitted to external servers.

Changes to This Policy

This privacy policy may be updated if new features are added. Any changes will be reflected on this page.
